# kid-fitness-pages

一個可直接部署到 GitHub Pages 的靜態展示頁，用來呈現小朋友的體適能測試結果。

## 功能

- 從 Excel 複製資料後直接貼到頁面
- 用下拉選單快速切換某位小朋友的雷達圖與摘要
- 一鍵下載所有學生的多頁 PDF，每位學生 1 頁，並依報表比例自動選擇頁面方向

## 檔案

- `index.html`: 頁面結構與 CDN 引用
- `style.css`: 版面與視覺樣式
- `app.js`: 資料解析、學生切換、雷達圖與 PDF 匯出

## Excel 貼上格式

第一列要是欄位名稱，至少要有 `姓名`，建議包含：

```text
學號	姓名	測試日期	柔軟度	肌耐力	心肺耐力	平衡協調	爆發力
1	王小明	2026-03-25	78	66	82	74	69
2	林小華	2026-03-25	85	72	77	81	75
```

規則：

- `姓名` 會被當成學生名稱
- `學號`、`編號` 會被辨識成學生編號
- `日期`、`測試日` 會被辨識成測試日期
- 其他數值欄位會自動當成圖表指標

## 使用方式

直接用瀏覽器開啟 `index.html` 即可預覽。

## GitHub Pages

如果這個資料夾本身就是一個 GitHub repo：

1. 把檔案 push 到 GitHub
2. 到 repo 的 `Settings > Pages`
3. Source 選擇要發佈的 branch
4. Root 選擇 `/ (root)`

## 備註

雷達圖適合展示已標準化後的分數。如果各測項單位不同，建議先建立換算規則，再放上圖表。
